О курсе

Имя лектора

Курс. Построение отчетов в “1С:Предприятие 8” с помощью системы компоновки данных (СКД).

Лекции

  • №1. Введение

    Рассматриваемые вопросы:
    1. Как проходит процесс обучения. Организационные вопросы.
    2. Чему мы научимся за время курса.
    3. Способы построения отчетов в “1С:Предприятие 8.3”.
      • Ручное формирование с последующим выводом в табличный документ;
      • Построитель отчетов;
      • СДК.
    4. Обзор технологии СКД.
    5. Пример разработки простейшего отчета.
    Результат лекции:

    Вам известны все популярные способы формирования отчетов в “1С:Предприятие 8.3” и вы прекрасно справились с задачей построения своего первого отчета при помощи технологии СКД.

  • №2. Как работает СДК. Разбираем по частям

    Рассматриваемые вопросы:
    1. Планируем построение отчета.
    2. Источники данных.
      • Запрос;
      • Объект.
    3. Связываем наборы данных.
    4. Параметры.
    5. Вычисляемые поля.
    6. Ресурсы.
    7. Вложенные схемы.
    Результат лекции:

    Вы познакомились с основными вкладками конструктора отчетов СКД. Разобрались с их предназначением и провели всю информацию на практике.

  • №3. Оформление. Настройка структуры

    Рассматриваемые вопросы:
    1. Как формировать структуру отчета.
      • Группировки;
      • Детальные записи.
    2. Настройка оформления.
      • Встроенные шаблоны оформления;
      • Условное оформление.
    3. Собственные шаблоны оформления.
    4. Варианты отчетов.
    5. Отборы.
    6. Детально разбираем вопросы вывода информации.
    Результат лекции:

    Оформление отчетов под вашим контролем. Вы умеете использовать встроенные шаблоны оформления и при необходимости знаете как полностью изменить типовое оформление.

  • №4. Диаграммы

    Рассматриваемые вопросы:
    1. Как добавить диаграмму в отчет на СКД.
    2. Добавление диаграммы из конструктора.
    3. Практикуемся выводить и настраивать диаграммы.
    Результат лекции:

    Вы знаете, что добавить диаграмму в отчет, созданном с помощью СКД - совершенно не сложно.

  • №5. Расчет итогов. Роли полей компоновки данных

    Рассматриваемые вопросы:
    1. Для чего нужны роли.
    2. Разбираемся с периодами.
    3. Нюансы настройки ролей для полей с остатками.
    4. Флаг “Игнорировать NULL”.
    5. Практикуемся.
    Результат лекции:

    Окно с “непонятными” настройками ролей больше не вызывает страха. Вы знаете в каких случаях следует выполнять настройку ролей и делаете это осознанно.

  • №6. Вложенные схемы

    Рассматриваемые вопросы:
    1. Что такое вложенная схема компоновки данных.
    2. Когда следует применять.
    3. Практика создания вложенной схемы.

  • №7. Язык выражений СКД

    Рассматриваемые вопросы:
    1. Знакомство с языком выражений СКД.
    2. Функции языка выражений СКД.
    3. Расширяем возможности. Подключаем функции общих модулей.
    4. Разбор типичных задач с применением языка выражений СКД.

  • №8. СКД + Встроенный язык программирования. Итоги

    Рассматриваемые вопросы:
    1. Зачем может потребоваться возможность составление схемы компоновки данных с помощью встроенного языка. Разбираем реальные кейсы.
    2. СКД для обработок.
    3. События в СКД.
    4. Практика. Закрепляем кейсы на практике.
    5. Итоги интенсива.
      • Что дальше?
      • Темы для самостоятельного изучения.
    Результат лекции:

    Вы умеете быстро и качественно формировать отчеты различной сложности с помощью системы компоновки данных. Вам больше не требуется придумывать “магические” методы для задач, которые эффективно решаются встроенными механизмами СКД.
